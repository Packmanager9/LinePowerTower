Power Towers

  • Click the black rectangle to start when ready
  • Build towers on the red lots
  • 50 points for green
  • 100 points for blue
  • Blue slows target 75%
  • Green deals 2.5X damage as blue
  • Don't let the dots reach the red zone
  • To replace a tower build the other color
  • To replace a tower build the other color in the same spot
  • To upgrade a tower build the same color in the same spot
